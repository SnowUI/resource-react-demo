## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install
# or
pnpm install
```

### Development

```bash
# Start development server
npm start
# or
pnpm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build

```bash
# Build for production
npm run build
# or
pnpm build
```
